# Epic Gaming Web Page

Yeah, u read that right. Welcome to this super clean, mini gaming landing page. It has flashy videos, slick scroll animations, and a clean dark mode vibe.

This whole project is heavily inspired by the absolutely insane Zentry gaming website. Go check out the real masterpiece here: https://zentry.com/

## What is Inside?

* Video Switcher: Click that big juicy NEXT button and boom—ur watching a new trailer. LOL it is that simple.
* Slick Scroll Animations: As u scroll down, headers blur nicely, cards pop up, and images expand to full screen. Modern CSS magic only.
* Fully Responsive: Looks great on ur desktop and ur phone. No broken layouts here, we fixed them all.

## The Files

* index.html — The skeleton holding all the sections together.
* style.css — The glow-ups, neon gradients, and scroll timelines.
* script.js — The brain responsible for swapping ur videos smoothly.

## How to Run It

1. Download the files.
2. Make sure u have ur videos and img folders set up with ur media.
3. Just double-click index.html to open it in ur browser. 

Yeah, no crazy terminal commands or installations needed. Have fun messing around with it.
